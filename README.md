# Proxy Collector (Сборщик прокси)

Программный комплекс для автоматизированного поиска, проверки и управления списком прокси-серверов. Проект включает в себя центральный веб-ресурс для дистрибуции модулей и локальное приложение с фоновой службой сбора данных.

## Особенности
- **Модульная архитектура:** Подключение новых источников через динамические пакеты (.dll).
- **Фоновый мониторинг:** Сбор данных о ресурсах системы (CPU, RAM) в реальном времени.
- **Интерактивная статистика:** Визуализация активности пакетов и работоспособности прокси.
- **Экспорт данных:** Сохранение результатов в формате CSV для внешнего использования.
- **Современный UI:** Интерфейс разработан с соблюдением принципов "Business Garden".

## Архитектура системы
Проект разделен на несколько ключевых компонентов:
1. **Центральный сайт:** Репозиторий пакетов и документации.
2. **Background Daemon:** Фоновая программа на C#, отвечающая за сбор и валидацию.
3. **Local Web UI:** Интерфейс управления на базе локального веб-сервера.
